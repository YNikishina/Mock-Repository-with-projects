# My Learning Projects

This repository contains examples of learning projects completed during my studies in Business Intelligence and Data Analytics at TripleTen.

| | | |
|-|-|-|
|__Project Name__|__Short Project Description__|__Tools Used__|
|[Sprint 1 Project: Analyzing the Manhattan Airbnb Market](https://github.com/YNikishina/Mock-Repository-with-projects/Sprint-1)|Analyzed Airbnb listings in Manhattan to identify the most attractive neighborhoods and listing types for short-term rental investment. The project focused on revenue potential, top-performing listings, and investment recommendations for potential investors.|*Google Sheets*|
|[Sprint 2 Project: SQL Database Analysis](https://github.com/YNikishina/Mock-Repository-with-projects/tree/main/Sprint-2)|Performed SQL analysis of a business database to answer analytical questions, identify patterns, and generate business insights. The project included joins, aggregations, subqueries, filtering, and KPI analysis using SQL queries.|*SQL*|
|[Sprint 3 Project: Conversion and Retention Analysis for an Online Store](https://github.com/YNikishina/Mock-Repository-with-projects/tree/main/Sprint-3)|Analyzed customer conversion funnels and retention metrics for an online store using spreadsheets. |[Sprint 4 Project: Profit and Returns Analysis for Superstore](https://github.com/YNikishina/Mock-Repository-with-projects/tree/main/Sprint-4)|Built Tableau visualizations to analyze profit centers, loss-makers, return rates, and business opportunities for Superstore. The project focused on identifying the most profitable segments and reducing losses caused by returns.|*Tableau*|
|[Sprint 5 Project: Return Rate Analysis Presentation for Superstore](https://github.com/YNikishina/Mock-Repository-with-projects/tree/main/Sprint-5)|Created a business presentation based on Tableau analysis of Superstore return rates. The project focused on presenting findings, explaining return drivers, and providing clear business recommendations for improving profitability.|*Tableau, Business Presentation*|
|[Sprint 6 Project: Shopify Apps and Reviews Analysis](https://github.com/YNikishina/Mock-Repository-with-projects/tree/main/Sprint-6)|Developed a Power BI dashboard to analyze Shopify apps, customer reviews, ratings, and market opportunities.
